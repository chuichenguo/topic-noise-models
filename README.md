# topic-noise-models
Please go to master branch. Main branch is only for readme.
## How to run
```
python TND.py
python NLDA.py
```
## If you want to check NPMI value
Paste your iteration result on the beginning of NPMI.py. Then run,
```
python NPMI.py
```
